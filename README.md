# Chess-Game---Python

Overview- 

This project is a final outcome of my culmiating project of coding any game in Computer Technology class. This chess game involves you playing a game of chess with real pieces, real movement and accurate rules agaisnt a bot or in a 2 player game. It is made with Python and Pygame to show a nice interactive chess board with special effects when moving the pieces. It has a turned based gameplay (White VS Black) and a detection tool when a check or a checkmate it given. Along with that, you can go back to your previous moves or quit a game. This game really builds your chess skills for a beginner and helps pro practice. In conclusion, this game satisfies the need of a realistic chess game against another opponent or a bot by showing the possible moves a user has when a piece is clicked to move.


How to run this-

Install Python and Pygame(pip install pygame)

Run this file (main.py)
