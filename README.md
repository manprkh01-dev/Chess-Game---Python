# Chess-Game---Python
My culminating project of coding any game you wish in Computer Tech. Class. This chess game involves 2 players or playing against a bot. Play this game and master your chess skills and enjoy the beauty of Chess.


♟️ Chess Game (Python + Pygame)

📌 Overview
This project is a fully functional chess game developed using Python and Pygame. It demonstrates understanding of game logic, object-oriented programming, and graphical interfaces.

The game allows two players to play chess locally with a visual board, piece movement, and rule enforcement or 1 player with AI.
🎮 Features
Interactive chessboard built with Pygame

Standard chess rules implemented

Piece movement validation

Turn-based gameplay (White vs Black)

Capture mechanics

Check detection (if implemented)

Clean and simple UI

🛠️ Technologies Used
Python 3

Pygame (for graphics and game loop)

🚀 How to Run
Make sure Python is installed on your computer.

Install Pygame if you don’t already have it:

pip install pygame

Navigate to the project folder:

cd chess-game
Run the game:

python main.py
